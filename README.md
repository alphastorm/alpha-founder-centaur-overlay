# Alpha Founder Centaur Qualification Overlay

This public repository contains the non-secret, qualification-only organization overlay used to exercise Centaur's supported overlay mechanism for Alpha Founder Gate B.

It contains:

- one read-only local attestation tool;
- one bounded workflow definition;
- one Claude Code skill;
- one sandbox system-prompt extension and one inert sandbox marker file;
- deterministic local tests.

The overlay contains no Alpha Founder product data, provider credentials, GitHub credentials, secrets, network clients, or external mutation capability. Qualification pins an immutable commit SHA; `main` is never a runtime selector.
